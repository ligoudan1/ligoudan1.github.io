3.深思

如何存储已经拖拽了的状态呢?

每次我们刷新页面,会返回默认的表格排序,然而我们想把我们已经拖拽完成的存储起来要怎么做呢?

这里就需要H5中的localStorage来帮忙了

1.监听拖拽改变事件
2.将变化了的表格顺序存储起来

localStorage.setItem("codelist", JSON.stringify(this.dropCol));
